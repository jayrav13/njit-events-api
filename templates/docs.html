Events at NJIT

About

This API was created to support the Events at NJIT iOS app at first. However, it only seems right that this API should be exposed for other NJIT developers to use as well!

Make a Request

The current endpoint for this API is http://eventsatnjit.jayravaliya.com/api/v0.2/events. A simple GET request will return to you up to 600 upcoming events at NJIT. The data is refreshed every Monday morning at 8:00am EST.

Response Details

The result will include high level statistics of the request:
{
	"completion_time": "0.695343971252",
	"count": 421,
	"current_date": "01/30/2016",
	"current_time": "05:37 PM",
	"response" : [...]
}
